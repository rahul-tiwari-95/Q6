HOW TO RUN TRAINING:
====================

1. Activate virtual environment:
   source venv/bin/activate

2. (Optional) Validate setup first:
   python validate.py

3. Start training:
   python main.py

4. Watch output - you'll see learning progress with metrics!

5. Training artifacts saved to:
   training_runs/<timestamp>/
   ├── checkpoints/     # Model files
   ├── logs/           # Detailed logs (JSON)
   └── plots/          # Training visualizations

KEY METRICS TO WATCH:
====================

1. Avg(100) - The moving average score
   - Should trend upward over time
   - Initially very negative (exploring)
   - Should improve as agent learns

2. Collected - Pellets collected per episode
   - Early: 0 (random exploration)
   - Mid: 1-2 occasionally
   - Late: 3-4 consistently

3. Caught - Times caught by enemies
   - Early: 3+ (frequently caught)
   - Mid: 2-3 (avoiding some)
   - Late: 0-1 (learning evasion)

4. ε (epsilon) - Exploration rate
   - Starts at 1.0 (100% exploration)
   - Decreases to 0.01 (99% exploitation)
   - Shows agent is learning strategy

EXPECTED CONVERGENCE TIMELINE:
==============================

Episodes 1-100:
- Very negative scores (exploring)
- 0 pellets collected
- Getting caught frequently

Episodes 100-500:
- Score improving slightly
- Occasional pellet collection
- Times caught decreasing

Episodes 500-1000:
- Score near 0 or slightly positive
- Regular pellet collection
- Better at avoiding initial catches

Episodes 1000+:
- Positive scores possible
- Consistent success
- Agent has learned strategy
