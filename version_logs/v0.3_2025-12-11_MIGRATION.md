# Version 0.3 Migration Log
**Date:** December 11, 2025  
**Type:** Major Strategy Change - Curriculum Learning Adoption

## CRITICAL DECISION: Strategy Migration

### What Changed
**Previous Approach (hunter_version_1.py):**
- Full difficulty from episode 1
- All 5 enemies present from start
- 2K episodes training
- No progressive learning
- **Result:** Agent completely failed (-1486 avg score, 0% win rate)

**New Approach (main.py - Curriculum Learning):**
- Progressive difficulty across 4 phases
- Enemies added gradually (2 → 3 → 4 → 5)
- 10K episodes with extended Phase 4 (5500 episodes)
- Learn fundamentals before facing full challenge
- Enhanced exploration (epsilon_min 0.05, decay 0.9995)
- Comprehensive visual analysis system

### Why This Change Was Made

**User's Philosophy:**
> "The env was too barren - and too fixed. The new implementation accepts common sense and logic as its friend - and we will make our agent work better -- I know it."

**Technical Rationale:**
1. **Curriculum Learning = Common Sense**: Humans don't learn complex tasks by diving into hardest difficulty. We build fundamentals first.
2. **Environment Too Hostile**: Throwing agent into 5 enemies with A* pathfinding from episode 1 is like throwing someone who can't swim into the ocean.
3. **Exploration Collapsed**: Old epsilon decay (0.995) meant agent stopped exploring by episode 500.
4. **No Pattern Time**: Old 2K episodes with full difficulty meant agent never got time to understand patterns.

**The Vision:**
- Agent needs time to learn patterns in manageable environment
- Gradually increase complexity as agent builds understanding
- Extended Phase 4 (5500 episodes) allows deep pattern emergence
- Better terminal feedback for watching training in real-time

### New Features in main.py

**1. Curriculum Learning (4 Phases):**
- Phase 1 (1500 eps): 2 Greedy bots - learn navigation
- Phase 2 (1500 eps): 2 Greedy + 1 Patroller - learn avoidance
- Phase 3 (1500 eps): 2 Greedy + 2 Patrollers - learn complex navigation
- Phase 4 (5500 eps): Full game - master complete challenge

**2. Enhanced Terminal Output:**
- Beautiful ASCII art borders
- Color-coded phase transitions
- Real-time metrics (score, pellets, caught, epsilon)
- Action distribution analysis
- Progress bars and visual separators
- Episode-by-episode tracking

**3. Comprehensive Analysis:**
- 5 detailed visualizations auto-generated
- Phase comparison charts
- Phase 4 deep dive (Early/Mid/Late analysis)
- Exploration strategy visualization
- Statistical summary dashboard

**4. Improved Hyperparameters:**
- Epsilon min: 0.01 → 0.05 (5x more exploration)
- Epsilon decay: 0.995 → 0.9995 (10x slower decay)
- Extended training: 2K → 10K episodes
- Phase 4 emphasis: 55% of total training time
